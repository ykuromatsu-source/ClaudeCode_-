# 次のステップ提案（CTA Examples）

このファイルは `insta-food-buzz` スキルが投稿文作成後にユーザーへ提案する「次のアクション」のフォーマット例です。

---

## 提案フォーマット

投稿文を出力した後、以下のような形式で次のステップを提示してください：

---

**▼ 次に何をしますか？**

以下から選んでお知らせください（複数選択OK）：

1. 📸 **別のメニューでも投稿文を作る**
   → 他の料理・ドリンクの情報を教えてください

2. 🎬 **ストーリーズ・リール用テキストも作る**
   → 同じメニューの短尺動画向けテキストを作成します

3. 🔄 **A/Bテスト用に別パターンを追加で出す**
   → 今回の3パターンとは違う切り口で2〜3案追加します

4. 🗓️ **投稿スケジュールの提案をもらう**
   → 週何回・何曜日・何時に投稿すべきかアドバイスします

5. 📊 **ハッシュタグを別のターゲット向けに変える**
   → 例：地域外へのリーチ拡大 / 外国人観光客向け 等

6. 🖊️ **他のSNS（TikTok・X・食べログ）向けに書き直す**
   → 媒体の特性に合わせてリライトします

7. 💬 **コメント返信テンプレートを作る**
   → 「どこのお店ですか？」「予約できますか？」などへの返信文

---

## 使い方のヒント

- ユーザーが選んだ番号に応じて、次のアクションをすぐに実行する
- 番号ではなく「ストーリーズも欲しい」などの自然な言い方でも受け付ける
- 提案は毎回全部出す必要はなく、文脈に応じて4〜6個に絞っても良い
