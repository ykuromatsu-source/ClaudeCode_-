# Instagram 飲食店投稿 スコアリング基準

このファイルは `insta-food-buzz` スキルが投稿文の品質を自己評価するための基準です。
投稿文を作成したら、以下の基準を満たしているか確認してください。

---

## 評価項目（各10点、合計100点）

### 1. 冒頭フック力（10点）
- [ ] 最初の1〜2行でスクロールを止められるか
- [ ] 「え、何これ？」「続きが気になる」と思わせるか
- 減点：「こんにちは！」「今日は〇〇を食べました」など平凡な書き出し

### 2. 五感の具体描写（20点）
- [ ] 食感（とろける・サクサク・もちもち等）が描かれているか
- [ ] 香り・風味・温度感が伝わるか
- [ ] 「美味しい」「最高」だけの抽象表現になっていないか
- 減点：空虚な形容詞の連発

### 3. 感情・共感の喚起（15点）
- [ ] 読んだ人が「行きたい」「食べたい」「保存したい」と感じるか
- [ ] 普遍的な感情（懐かしさ・驚き・幸福感・罪悪感）に触れているか

### 4. 写真との連動性（15点）
- [ ] 写真の特徴（色・断面・チーズ伸び等）が文章に活きているか
- [ ] 写真を見ていない人でもビジュアルが想像できるか

### 5. ストーリー性・独自性（10点）
- [ ] 他の飲食店でも使えるような汎用文になっていないか
- [ ] 「この店・このメニューだからこそ」の要素が入っているか

### 6. 読みやすさ・テンポ（10点）
- [ ] 適切な改行・絵文字で視覚的に読みやすいか
- [ ] 一文が長すぎず、スマホ画面で読みやすいか

### 7. 行動誘導（CTA）の自然さ（10点）
- [ ] 「保存して」「来てね」などが不自然に押しつけがましくないか
- [ ] 最後の一文がポジティブな余韻を残しているか

### 8. ハッシュタグの適切さ（10点）
- [ ] 大・中・小の規模感が混在しているか
- [ ] 30個以内に収まっているか
- [ ] ジャンル・地域・メニュー名を含む発見されやすいタグか

---

## 合格ライン

- **80点以上**：バズ狙いに十分な水準
- **70〜79点**：改善推奨（特に低スコア項目を修正）
- **70点未満**：要リライト

---

## よくある失点パターン

❌ 「〇〇が美味しかったです！またリピートします！」→ 情報ゼロ
❌ 「#グルメ #食べ物 #飯テロ」だけのビッグタグ集め → 埋もれる
❌ 冒頭が店名から始まる → スクロールされる
❌ 一文が80字以上続く → スマホで読まれない

✅ 「熱々の〇〇を割ると、中からとろとろの〇〇が…」→ 五感×ビジュアル
✅ 「#渋谷カフェ巡り #チーズタッカルビ東京 」→ ニッチタグで発見される
